# Typecho-theme-DUX

>Forked from https://github.com/hiCasper/Typecho-theme-DUX

Typecho-theme-DUX 是一款基于大前端DUX for Wordpress主题修改而来的主题。

我个人在 Typecho-theme-DUX 的基础上，修改了样式，没改功能

样式来自于WP主题- [Yimik]()
